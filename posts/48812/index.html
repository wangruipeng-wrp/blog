Nginx 配置

发表于 2021-08-17 | 分类于 Linux |
| 字数统计: 807

Nginx 的配置文件都放在 conf 目录下，其中最重要的就是 nginx.conf 配置文件，本文也主要记录该配置文件中的配置

为了使配置文件能够更方便管理，把一些站点配置写到外面来，不要直接写入 nginx.conf 文件

在 conf 文件夹下创建一个 hosts 文件夹，之后将每个站点的配置都以 域名.conf 的命名格式保存在 hosts 文件夹中

之后在 nginx.conf 文件的 http 的大括号内引入自己写的配置文件 include hosts/*.conf;

传输配置

日志格式：

名称 解释
$remote_addr 客户端IP
$remote_user 远程客户端用户名，一般为：’-‘
$time_local 时间和时区
$request 请求的 url 和 method
$status 响应状态码
$body_bytes_send 响应客户端内容字节数
$http_referer 记录用户从哪个链接跳转过来的
$http_user_agent 用户所使用的代理，一般情况下为浏览器
$http_x_forwarded_for 通过代理服务器来记录客户端的IP

虚拟主机配置

location 的匹配规则：

  • 空格：普通匹配
  • =：精确匹配
  • ~*：匹配正则表达式，不区分大小写
  • ~：匹配正则表达式，区分大小写
  • ^~：以某个字符路径开头
